# homebrew-fusiondatacli

Homebrew tap for [FusionDataCLI](https://github.com/schneik80/FusionDataCLI) — a terminal browser for Autodesk Platform Services Manufacturing Data Model.

## Install

```sh
brew install schneik80/fusiondatacli/fusiondatacli
```

## Update

```sh
brew upgrade fusiondatacli
```
